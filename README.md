# 🌾 Agribusiness Origination Dashboard

Dashboard executivo de inteligência comercial para gestão de originação de grãos (milho/soja).

## ✨ Funcionalidades

- 📊 Panorama Geral com KPIs
- 🗺️ Mapa de Exposição Geográfica
- 📦 Inteligência Logística com Simulador de Frete
- ⚠️ Alertas Inteligentes
- 📈 Previsões ML
- 📋 Terminal Bloomberg
- 📥 Exportação Excel/PDF

## 🚀 Deploy

Clique no botão abaixo para fazer deploy no Streamlit Cloud:

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://share.streamlit.io/new)

## 📋 Requisitos

- Python 3.9+
- Dependências em `requirements.txt`

## 🏃 Executar Localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📊 Dados

Dashboard inclui dados de exemplo com 300 contratos de originação.

## 🎨 Design

Executive Dark Mode com cores neon (verde para soja, dourado para milho).
